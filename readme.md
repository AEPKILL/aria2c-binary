# ariac2-binary

Provide Aria2c multi-platform binary files

## description

This project provides aria2c binary files for the following platform:

- windows
  - [x] x86
  - [x] x64
- macOS
  - [x] x64
  - [x] arm64
- linux
  - [x] x64
  - [x] arm64
